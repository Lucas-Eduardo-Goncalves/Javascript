Aritmeticos

Os operadores aritmeticos são usados para fazer contas, no javascript

São eles:

  • 5 + 2 = 7
  • 5 - 2 = 3
  • 5 * 2 = 10
  • 5 / 2 = 2.5
  • 5 % 2 = 1
  • 5 ** 2 = 25
